# CMS_CLOUD_DA-TEST-SEM-5

Cube of number get/post request app. (For CMS Lab VIT)

<br>

Steps followed <br>

```npm init```<br>

```npm install express body-parser```<br>

```npm install nodemon```<br>

To run the app : <br>

```nodemon server.js```<br>

The app runs at **localhost:3000**
